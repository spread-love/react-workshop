Bem vindo ao primeiro workshop React Hands ON! Esse workshop foi criado com o objetivo de espalhar o conhecimento de React na nossa comunidade. Ele é direcionado para pessoas que tem conhecimentos básicos de JavaScript, HTML e CSS.

Vamos começar dos assuntos mais básicos, como o que é o React e o que ele faz, até assuntos mais avançados como Redux e separação de código. **Não é esperado que você tenha conhecimento sobre React previamente**.
